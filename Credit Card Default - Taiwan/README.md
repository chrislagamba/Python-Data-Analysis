## Introduction
This is an analysis of a dataset that is "aimed at the case of customers' default payments in Taiwan and compares the predictive accuracy of probability of default among six data mining methods."

## Dataset
The dataset was obtained from the UCI Machine Learning Repository here: https://archive.ics.uci.edu/dataset/350/default+of+credit+card+clients (link last checked 01 August 2025).

There are a total of 30,000 instances with 23 features per instance. There are no missing values in this dataset per the description.

Variable information was available on the UCI Machine Learning Repository where the dataset was obtained.
